# 介绍

是一个基于 Vue 3 的现代化组件库，提供了一系列美观且实用的 UI 组件。

## 特性

- 🚀 基于 Vue 3 和 TypeScript 开发
- 💪 完整的类型支持
- 🎨 支持主题定制
- 📦 支持按需引入
- 📱 响应式设计

## 安装

::: code-group

```bash [npm]
npm install lcz-kit
```

```bash [yarn]
yarn add lcz-kit
```

```bash [pnpm]
pnpm add lcz-kit
```

:::

## 基础使用

```vue
<script setup>
import { Button } from 'lcz-kit'
</script>

<template>
  <Button>点击我</Button>
</template>
```
